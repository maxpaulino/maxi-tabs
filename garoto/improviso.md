## Notation Key

* **h** = Hammer-on
* **p** = Pull-off
* **b** = Bend
* **/** = Slide up
* **\\** = Slide down
* **x** = Mute / dead note
* **(n)** = Ghost note / optional note
* **L** = Tied note
* **g** = Grace note
* **s** = Slide
* **NH** = Natural harmonic
* **^** = Brush with the thumb
